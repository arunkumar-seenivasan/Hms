# Hotel Management System

### Introduction

The Hotel Management System is a desktop application developed using Python, Tkinter, and SQLite. It helps manage hotel rooms, bookings, and checkouts in a simple and efficient way.

### Objectives

* Manage room details
* Allow customers to book and checkout rooms
* Provide secure admin access
* Store data using a database

### Technologies Used

* Python
* Tkinter (GUI)
* SQLite (Database)

### Features

* Admin login to add rooms
* Customer room booking and checkout
* View room availability
* Automatic update of room status
